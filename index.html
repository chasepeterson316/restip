What is EconValues ?

EconValues is a test created by Solar#5896 that attempts to measure desired economic systems on 8 axes.

What are the 16 values ?

There are 8 independent axes - Incentive, Structure, Intervention, Centralisation, Technology, Land, Inheritance and Labor - and each has two opposing values assigned to them :

Equity ( Incentive )

Competition ( Incentive )

Horizontal ( Structure )

Hierarchy ( Structure )

Demand ( Intervention )

Regulation ( Intervention )

Central ( Centralisation )

Local ( Centralisation )

Automation ( Technology )

Employment ( Technology )

Commons ( Land )

Rent ( Land )

Birthright ( Inheritance )

Merit ( Inheritance )

Unionized ( Labor )

Divided ( Labor )
